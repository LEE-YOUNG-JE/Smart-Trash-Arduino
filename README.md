# Smart-Trash-Arduino

설계 내용
##sensor Board(smart trash can) HW


a. IO cube

	D0:버튼
	D2:LED
	A0:조도센서
	A1:가변저항(중량계로 치환)
b. 부저

c. 터치센서(L,R)

d. led matrix

e. RGB led

f. GPIO 15번 핀

g. wifi module


##main Board(분리수거장 관리 보드) HW

a. real cube
	- 온습도, 압력 센서
	- 마이크 센서	
b. 부저
c. led matrix
d. RGB led
e. GPIO 15번 핀
f. Bluetooth module
g. Serial port
